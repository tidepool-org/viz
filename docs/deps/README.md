## Dependencies

Key dependencies used in this repository for data visualization:

- [D3](https://d3js.org/ 'D3: Data-Driven Documents') ([read more](./D3.md)) - scales, axes, and data transformations
- [React](https://facebook.github.io/react/ 'React') ([read more](./React.md)) - UI components
- [react-motion](https://github.com/chenglou/react-motion 'GitHub: react-motion') ([read more](./ReactMotion.md)) - animations

Follow the "read more" links to learn about our specific usage patterns.
